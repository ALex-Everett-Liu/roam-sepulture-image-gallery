Gallery Management

How to manage your gallery:

1. Load Data: Use the Settings panel to load JSON files with image data

2. Browse Files: Use the file browser to select JSON files from your system

3. Image Groups: Create collections with major images and subsidiary images

4. Import/Export: Use the buttons above to manage your gallery data

5. Structure: Each image/video needs: title, description, src (for major images), and display properties

6. Manual Sizing: Each image uses direct 'width' and 'height' properties for precise control - no complex grid system

7. Video Support: Videos (.mp4, .webm, .mov, .avi, etc.) are automatically detected and displayed with controls

JSON Format Example 
{
  "images": [
    {
      "id": 1,
      "title": "Sunset Beach Collection",
      "description": "Beautiful sunset at the beach with multiple perspectives",
      "src": "path/to/your/image.jpg",
      "ranking": 9.2,
      "tags": ["sunset", "beach", "ocean"],
      "width": "600px",
      "height": "400px",
      "isMajor": true,
      "groupId": "group-1",
      "date": "2025-01-26T16:45:00.000Z"
    },
    {
      "id": 2,
      "title": "Nature Video",
      "description": "Beautiful nature video with wildlife",
      "src": "assets/nature.mp4",
      "ranking": 8.5,
      "tags": ["nature", "video", "wildlife"],
      "width": "1200px",
      "height": "600px",
      "isMajor": true,
      "groupId": "group-2"
    },
    {
      "id": 3,
      "title": "Sunset Beach - Wide Shot",
      "src": "path/to/subsidiary1.jpg",
      "width": "400px",
      "height": "150px",
      "isMajor": false,
      "groupId": "group-1",
      "majorImageId": 1
    }
  ]
}
Major Images: Include ranking, tags, description (used for filtering/sorting)
Subsidiary Images: Inherit metadata from major image, only need src, width, height
Standalone Images: Set isMajor: true without groupId for individual display
Date Field: Automatically added when importing new images (ISO 8601 format)
Manual Layout: Each image uses 'width' and 'height' properties for exact sizing control
Flexbox System: Gallery uses flexible layout - images arrange naturally based on their specified sizes
Sorting: Can sort by date added (oldest/newest first) in addition to ranking and name
